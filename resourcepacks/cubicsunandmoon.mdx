You looked at the Moon and you thought: "why isn't it also a cube?"
Well, then this Resource Pack is perfect for you! It recreates the Minecraft Sun and Moon in 3D!

## Features

   - Cubic Moon texture with 8 phases.
   - Cubic Sun texture.
   - Support for all Minecraft versions and Mods.

## How to Install
- Download the Featured Version.
- Open Minecraft and go to Options > Resource Packs / Texture Packs > Open Pack Folder.
- Copy/Cut and Paste the file downloaded there.
- Activate the Resource Pack.  

<Note>If it appear as Incompatible, don't worry, it should work.</Note>

## Support
### Minecraft versions
This resource pack supports all Minecraft versions that allow texture/resource packs to be used, from Alpha 1.2.2 to 1.21. It includes textures for April Fools updates as well.

### Shaders
It also works with shaders that do not change the Sun and Moon texture or have an option to switch the Sun/Moon style. For example it works with:

   - BSL Shaders
   - Complementary Shaders.

### Mods
It also offers support for mods that add sky features, such as planets and moons:

   - [Blue Skies](https://www.curseforge.com/minecraft/mc-mods/blue-skies) by Modding Legacy
   - [Ad Astra](https://modrinth.com/mod/ad-astra) by Terrarium
   - [Galacticraft](https://modrinth.com/mod/galacticraft-legacy) by TeamGalacticraft
   - Any mod that uses the Vanilla texture.
