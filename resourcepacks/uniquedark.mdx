## Unique Dark (Free)
 Free version of Unique Dark includes a few textures and basic features like More Readable Settings, for you to get understanding of the pack, so you could decide if you want the full version or not.
Included textures

    - Inventory
    - Crafting table
    - Furnace
    - Chest
    - Enchanting table
    - Anvil
    - Shulker box
    - Recipe book
    - Hotbar
    - Buttons
