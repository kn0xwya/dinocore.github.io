## About
This is a light-weight resourcepack mainly provides vanilla-look 3D models.

## Features
    - Minecraft version support from 1.8 to latest (packs for Minecraft below 1.14 can only found in [github page](https://github.com/GeForceLegend/Minecraft-3D-Default); packs for snapshots can be manually downloaded from github)

    - 3D Models with vanilla-look textures;

    - Optifine Custom Entity Models(not support 1.8 with Optifine version below L5);
